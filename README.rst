SIMP Core
=========

This is the build root for the 5.2 series of `SIMP`_.

Supported releases
------------------

This branch supports:
  * `RHEL 7.2`_
  * `CentOS 7 1511`_

Getting Started with SIMP
-------------------------

Please reference the `SIMP Getting Started Guide`_ for information on
getting started with a SIMP build.

Getting Help
------------

To get help with SIMP, please check the `Help Documentation`_ or visit the
`SIMP Homepage`_ for additional information.
